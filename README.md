# 项目介绍
ZK可视化工具

# 项目简介
- 后端：用springboot搭建框架,日志使用aop切面方式实现
- 前端：基于bootstrap搭建基本页面，表格使用bootstrap-table实现，zk树用ztree实现，弹层使用layui


# 项目功能
- zk连接管理
    - 可对zk连接进行管理，包括新增，删除，修改，建立连接，关闭连接操作
- zk节点管理
    - 当建立zk连接后，展示zk树
    - 点击树节点加载树的子节点，点击节点，在右侧视图中可查询节点详细信息
    - 节点上有操作按钮，可对节点进行新增，删除，修改
- 操作日志查询
    - 对zk的操作进行可视化的查询
